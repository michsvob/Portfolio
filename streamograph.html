Description

The point of this project was to make an automatically updated dashboard presenting the development of scrap products in my company. In my company, data on scrap products is manually exported from SAP into MS Excel on a daily basis. First, I wrote an Excel macro which produces a JSON file of the data. Then, I programmed a JavaScript code relying on a d3 library and interactively visualising the data within a styled html file. I have reproduced the report below using sample data.
